git push到github上遇到的一些问题

Dec 24 2022

问题描述#

git上传blog到github上时，遇到一些问题

执行

git remote add origin https://github.com/xxx/xxx.github.io.git

powershell返回

error: remote origin already exists.

问题原因#

git中已经有origin了，需要先解除已有origin再添加新的origin

解决方法#

  1. git remote rm origin 解除origin
  2. 再运行git remote add origin https://github.com/xxx/xxx.github.io.git
